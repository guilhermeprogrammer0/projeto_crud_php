PHP + MYSQL

CRUD é o acrônimo para Create (criar), Read (ler), Update (atualizar) e Delete (apagar). Com essa explicação, já dá para intuir que o CRUD é uma sequência de funções de um sistema que trabalha com banco de dados, seja ele na sua máquina ou na nuvem. É um conjunto de operações que pertence ao agrupamento chamado de Data Manipulation Language (DML) ou, em português, Linguagem de Manipulação de Dados.
